¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido en la web.








Caracteristicas

- Estructura básica: Comienza con !DOCTYPE html seguido por etiquetas html, head, y body.

- Etiquetas comunes: Incluyen encabezados h1 - h6, párrafos p, imágenes img, enlaces a, listas ul, ol, y tablas table.

- Atributos: Añaden información extra a las etiquetas, como src en img para definir la fuente de una imagen o href en a para un enlace.

- HTML5 y semántica: HTML5 introdujo etiquetas como header, footer, nav, article, que agregan significado y mejoran la accesibilidad y SEO.
