¿Qué es JavaScript?

JavaScript es un lenguaje de programación de alto nivel utilizado para crear interactividad en las páginas web, siendo uno de los tres pilares básicos del desarrollo web junto con HTML y CSS. Originalmente desarrollado para añadir dinamismo a los sitios web, JavaScript es actualmente una tecnología fundamental tanto en el frontend como en el backend, ampliando su alcance más allá del navegador.
