¿Qué es CSS?

CSS (Cascading Style Sheets) se utiliza para estilizar el contenido HTML.

Caracteristicas

- Separación de contenido y presentación: CSS permite mantener el HTML para la estructura y el contenido, mientras que el estilo y diseño se gestionan en un archivo CSS separado. Esto facilita el mantenimiento y la actualización de los estilos de un sitio web sin tocar el contenido.

- Selectores flexibles: CSS ofrece una amplia variedad de selectores (de etiquetas, clases, IDs, pseudoclases, y atributos) para aplicar estilos específicos a elementos en una página, permitiendo un control preciso y dinámico.

- Estilos en cascada e inherencia: La cascada de CSS decide qué estilos aplicar si hay conflictos entre reglas. Además, los estilos pueden heredarse, lo que significa que ciertos elementos reciben estilos de sus elementos principales automáticamente.
